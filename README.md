TaskFlow is a dynamic & collaborative full-stack task management app developed using React.js for the frontend, BootStrap for the UI/UX components, Express.js for API development, MongoDB for the database, and Node.js for the runtime environment. This app is particularly geared towards other students at the University of Waterloo, with features dedicated to Learn and Marmoset specific content, helping you to manage your submissions all in one place.
